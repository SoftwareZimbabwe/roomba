I use two methods to clean tiles:

Natural Method

I go to the nearest tile row by row, and save battery on unnecesary moves by going to the next rwo once the current is cleaned. Using an 8 x 8 tiled floor.

Predictive Method More efficient

I first think and decide on a path then walk that path in the order that use the least moves. This, however, consumes quite some CPU power because of the many combinations available, so for this example, I am showing you a 5 x 5 tiled floor.
